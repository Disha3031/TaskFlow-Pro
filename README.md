# 🚀 TaskFlow Pro - Boss Battle Edition

[![Live Demo](https://img.shields.io/badge/Live-Demo-brightgreen?style=for-the-badge)](https://disha3031.github.io/TaskFlow-Pro/)

TaskFlow Pro is a fully responsive, highly gamified task management web application designed to turn your daily to-do lists into an engaging RPG arcade loop. Instead of just checking off chores, you break down goals into sub-quests to damage epic boss monsters, earn Gold, accumulate XP, and level up your productivity rank!

🔗 **Live Link:** [https://disha3031.github.io/TaskFlow-Pro/](https://disha3031.github.io/TaskFlow-Pro/)

---

## 🎮 Key Features

* **⚔️ Dynamic Boss Fights:** Turn major projects into an "Epic Quest". Complete your sub-quests to deal damage to monsters like the *Procrastination Kraken* or the *Due-Date Demon*.
* **⏱️ Real-time Urgency Tracker:** Calculates exactly how many hours/minutes are left until midnight of your task due date, automatically flagging late items with an active `🚨 Overdue!` tag.
* **📈 Priority Scaling Rewards:** Higher priority items scale up your outputs—dealing more boss damage and awarding extra XP ($+30\text{ XP}$ for High, $+20\text{ XP}$ for Medium, $+10\text{ XP}$ for Low).
* **🔄 Infinite Loop Progression:** Defeating a monster triggers a victory screen, updates your currency wallet ($+100\text{ XP}$, $+50\text{ Gold}$), and procedurally unlocks the next level with increased health stakes.
* **🌓 Dark & Light Modes:** Includes a smooth UI state toggle adapting color palettes for early-morning planning or late-night gaming sessions.

---

## 🛠️ Technologies Used

* **HTML5:** Structural architecture, form fields, and modal overlay configurations.
* **CSS3:** Custom layouts using CSS Grid and Flexbox, animation keyframes (`shake`, `flyLeft` particles), and CSS variable systems for global dark-mode state management.
* **JavaScript (ES6+):** Application state management, calculation loops for date deltas, dynamic DOM painting pipelines, and modulo arrays for the endless boss generation engine.
